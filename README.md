# WhatSpyglass
*A small Fabric mod that replaces the vanilla spyglass experience with the WHAT meme template.*

This mod shows the WHAT meme template, plays the full EarthBound "Guardian Battle" theme, and hides parts of the HUD while zoomed in. It delivers the meme in a cleaner format compared to that of a resource pack.

## Supported versions
This project is structured with separate branches for different Minecraft versions instead of a single multi-version setup.

Fabric loader greater or equal to 0.18.4.
- Latest development is always on the ```main``` branch.
  - Currently: **1.21.6 - 1.21.11**
- Older versions are available under the ```archive/``` branches
  - Archive branches Include: **1.19 - 1.21.5**

Bug fixes may be applied across multiple branches where relevant.

## Version Notes
In some branches, you may need to manually adjust version-related values before building:
- ```gradle.properties```
- ```fabric.mod.json```

This is mainly for mappings / API differences between Minecraft versions.
The actual source code in each branch should already be correct.
